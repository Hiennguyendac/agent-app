# MVP Scope

## Objective

Build a simple working version of Agent App.

## In Scope

- create task from dashboard
- store task
- route task to one agent
- return a simple output
- show output on screen

## Out of Scope

- real publishing
- real email sending
- advanced security
- billing system

## Task Types

1. Growth task
2. Ops task
3. Optimizer task
4. Sales task
